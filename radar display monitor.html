HOW TO USE THIS FILE

  1. Open this file in Google Chrome or Microsoft Edge (required for Web Serial).
  2. Upload radar_esp32.ino to your ESP32 using Arduino IDE.
  3. Connect your ESP32 via USB.
  4. Click Connect ESP32 above and select the correct COM port.
  5. The radar will switch from simulation to live data automatically.
  6. If you just want to preview the radar without hardware, it runs in simulation mode by default.

Serial format expected from ESP32: ANGLE:90,DIST:45 at 115200 baud.
